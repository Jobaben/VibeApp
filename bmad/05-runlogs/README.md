# Runlogs

This directory contains session logs for all BMAD role sessions.

## Naming Convention

```
YYYY-MM-DD-HH-MM-<role>-<summary>.md
```

Example: `2025-01-15-14-30-analyst-initial-brief.md`

## Log Contents

Each runlog should include:
- Session timestamp
- Role activated
- Input artifacts referenced
- Output artifacts created/modified
- Key decisions made
- Issues encountered
- Next steps recommended
